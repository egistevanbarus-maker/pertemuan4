## 🛠️ Logic Implemented
- **useState Hook:** Mengelola state `count`, `nama`, dan `bgColor`.
- **Event Handlers:** Tombol (+) dan (-) untuk counter, serta `onChangeText` untuk input nama secara real-time.
- **Validation Logic:** Counter tidak bisa minus (tidak boleh < 0).
- **Dynamic Background:** Tombol "Ganti Warna Background" untuk mengubah warna background aplikasi.
